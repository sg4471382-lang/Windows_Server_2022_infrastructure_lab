## 🌟 Overview

This repository contains a step-by-step guide and practical implementation details for setting up a complete Windows Server 2022 infrastructure. Created by Sawan Gupta (Batch - Super 30 offline), this project covers the end-to-end process of deploying a server environment from scratch, configuring core enterprise network services, and managing client-server architecture in a virtualized environment.

## ✨ Key Features & Configurations

* 💻 **Virtual Environment Setup:** Step-by-step configuration of Oracle VirtualBox for both Windows Server 2022 (Desktop Experience) and Client OS, including bridged network adapter settings for seamless connectivity.


* 🏢 **Active Directory Domain Services (ADDS):** Promotion of the server to a Domain Controller and establishment of a new forest.


* 🌐 **Core Network Services:** Configuration of integrated DNS and setup of a DHCP server with defined IP address scopes.


* 🔐 **Group Policy Management:** Implementation of domain-wide security policies, specifically enforcing password complexity rules using Group Policy Objects (GPO).


* 👤 **User & Access Management:** Provisioning administrator rights to local users and verifying domain access from client machines.


* 🛠️ **Enterprise Server Roles:**
* 🖨️ **Print Server:** Centralized network printer management.


* 🌍 **Web Server (IIS):** Installation and configuration of Internet Information Services.


* 📧 **Mail Server:** Implementation of messaging services and user mailboxes using MailEnable.




* ⚙️ **Post-Installation Administration:** Basic server configuration, including static IP assignment, firewall verification, and remote desktop enablement.



## 🛠️ Technologies & Tools Used

* 🪟 **Operating System:** Windows Server 2022 Standard Evaluation (Desktop Experience).


* 🖥️ **Virtualization:** Oracle VirtualBox 7.2.4.


* 💾 **Bootable Media Tool:** Rufus.


* ✉️ **Third-Party Software:** MailEnable (for mail server configuration).



## 📂 Project Documentation

The complete methodology, including visual screenshots of the configuration wizards, GPO settings, and network properties, is documented within the project files. This serves as a practical reference for replicating a standard small-to-medium enterprise IT infrastructure.
